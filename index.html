Dans quel but et à qui ça s'adresse

Notre application vous fait une analyse de votre activité sur les réseaux sociaux. Enfin, nous voulons vous faire prendre conscience de ce que vous regardez, tout en vous proposant de nouvelles recommendations pour faire varier votre contenu.

Notre application s'adressera à toutes les personnes voulant savoir quel est son niveau de diversification, pour pouvoir controler et réguler son contenus

Protection de vos données personnel

La protection de vos données est notre préocupation majeure pour notre projet. Nous vous promettons une transarence total lors de vos premiers pas sur notre application.

Une connexion simple via une génération d'un username et un mot de passe sous la forme d'un code PIN, vous permettant une authentification minimum.

Une suppression automatiques de votre profil s'executera après un an d'inactivité sur notre application.

Traitements des données

Les données de vos application (hashtag, titre de vidéo, ect), seront analysés, classé et stocké.

Ces dernières nous permettrons de vous faire une classification simple de votre activité.
